I added multiple features

The first is that the whole assignment was written using Bootstrap as a basic html frontend
so that its prettier and scales. It also allows for me to have editable colorpickers

I also added multiple iterations per render, this means you can have an iteration each with 
different colors. It also has a randomize button allowing for a completely randomized
iteration.

TO GENERATE 10 ANIMATIONS WITH DIFFERENT SCALES AND COLORS YOU NEED TO CLICK THE ANIMATE
BUTTON.

THE ANIMATION MAY WAIT FOR A SECOND. THIS IS DUE TO THAT ITS NOT FRAME DEPENDENT BUT
TIMER DEPENDENT. IF IT WAITS FOR A SECOND OR TWO, THAT IS TO BE EXPECTED.

another feature is the scaling sliders. the scaling sliders allow you to scale the pre-
rendered image by an amount. you just slide the slider and it re-draws the scaled image.

THERE IS A REMAINING BUG, but its odd. If you do the animation once its fine. But if you
then do it again, for some reason it stops displaying on the canvas. Its definitely doing
something as i step through the code, but i cannot figure it out.
